### 課題１

- Markdown形式で、説明文をつけながら掛け算と割り算をやってみよう

Markdownの中で解析ソフトに計算してもらうために**コードチャンク (`{r}`)
を挿入** するには、Windowsでは( `Ctrl` + `Alt` + `I` )、Macでは( `Cmd` +
`Option` + `I`)を押すと、コードチャンクが挿入されます。

- チャンクの中で $(5+4) \div 3 - 2 \times 1$ [^1]を計算するにはどう表現するか？

[^1]:参考ですが、数式はTeXと言って、Markdownの中で`$`で囲むと数式として表現できます。

- ついでに、Markdownの`##`の数[^2]を変えて、見出しの大きさを変えてみましょう

[^2]:生成AIに指示を出す時に、`#`をつけてから書くとAIが理解しやすいと言われていますが、Markdownの中で`#`は見出しを表すために使われます。`#`の数が多いほど見出しの大きさが小さく(階層が下に)なります。

## 数字ではないため、ってどういうこと？

データの中身を確認する

``` r
str(x)
```

    ##  num 1

``` r
str(a)
```

    ##  chr "not"

イメージ的には、numは「数」、chrは「文字」という意味で、Rはデータの中身を見て、数なのか文字なのかを判断していると考えるとわかりやすいかもしれません。<br>

（ちなみに、intは整数、dblは小数点を含む数、logiはTRUE/FALSEを表す論理値と言います。）

## 課題３（便利なショートカットキーの一部）

多分機能を読んでもわかんないだろうから、全部自分で試してみて、何が起こるかを見てみよう。

| 機能 | Windows / Linux | macOS |
|:---|:---|:---|
| **現在の行を実行** | `Ctrl` + `Enter` | `Cmd` + `Enter` |
| **コメントアウト（解除）** | `Ctrl` + `Shift` + `C` | `Cmd` + `Shift` + `C` |
| **コードチャンク (`{r}`) を挿入** | `Ctrl` + `Alt` + `I` | `Cmd` + `Option` + `I` |
| **全ファイルから検索** | `Ctrl` + `Shift` + `F` | `Cmd` + `Shift` + `F` |
| **コードの自動整形** | `Ctrl` + `Shift` + `A` | `Cmd` + `Shift` + `A` |
| **ショートカット一覧を表示** | `Alt` + `Shift` + `K` | `Option` + `Shift` + `K` |
| **変数名の一括置換** | `Ctrl` + `Alt` + `Shift` + `M` | `Ctrl` + `Option` + `Shift` + `M` |
| **チャンク全体を実行** | `Ctrl` + `Shift` + `Enter` | `Cmd` + `Shift` + `Enter` |
